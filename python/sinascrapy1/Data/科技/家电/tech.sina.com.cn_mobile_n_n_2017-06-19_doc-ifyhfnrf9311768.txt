　　2017年E3游戏展前，微软发布了史上最强的游戏主机Xbox One X。虽然智能机与游戏主机之间并没有直接联系，但“最强”与“One X”两个词，却勾起了我的些许回忆。认真回想一下，我真正拥有的第一台智能手机，就是HTC One X。　　HTC One X是一部很有意义的手机，它是HTC辉煌的顶点，也是HTC手机衰落的开始。自HTC One X之后，HTC依旧推出了许多优秀的产品，但遗憾的是，这些产品大多叫好不叫座。那么，HTC是否还有未来，HTC的手机业务是否能起死回生呢？　　　　或许你已经听过很多关于HTC在安卓领域辉煌的开始，也一定记得雪姨（王雪红，HTC董事长）说过的那句“若未来手机厂商只剩下两家，HTC一定是其中一家”。那时，HTC推出了全球第一款安卓手机HTC G1，推出了华丽的Sence UI。而Nexus One的诞生，更是将HTC在安卓阵营的影响力推到了一个全新的高度。　　来到2011年，对于HTC而言，这是激进的一年。这一年，HTC推出了第一款裸眼3D手机HTC EVO 3D。尽管放在今天来看，裸眼3D与背部双摄都算不上什么新鲜事物，但当时这款手机对消费者的震撼是难以想象的。甚至有人评价道，在2011年真正能决定高端产品形象的并不是苹果，而是HTC。　　2011年HTC还做了一件事，其收购了Beats的多数股权。由此直接导致在很长一段时间内，部分消费者将搭载了Beats Audio的HTC手机当做是真正的音乐手机。可以说那时的HTC，不仅是技术的代表，更是潮流的代表。　　如果按照这个势头继续，HTC本来不会是现在的模样。　　　　进入2012年，HTC开始在旗舰机上尝试全新的技术。HTC One X就是在这样的背景下诞生的。一体化的聚碳酸酯机身，强劲的四核芯片，4.7寸的大屏加上一贯优秀的Sence UI，HTC One X本应当是一款优秀的，且不会失败的产品。　　但现实很残酷，HTC One X不仅没能为HTC带来荣耀，反而将HTC拖入了深渊。　　HTC One X自上市之初，就饱受屏幕黄斑与机身过热的困扰。HTC One X的屏幕黄斑，有些是由于手机屏幕出厂时的贴合问题，有些则是手机运行过程中过热所导致。较为玄学的是，黄斑出现的时机是未定的，能否消失也是未定的。　　机身过热的问题，大多人认为是HTC采用的Tegra芯片造成的。不得不承认的是，Tegra系列芯片的性能冠绝群雄，但随之而来的发热与功耗问题却始终没能找到合理的解决方案，或许这就是Tegra芯片最终退出智能机市场的原因吧。　　HTC One X同年的对手，一款是三星的一代神机Galaxy S3，一款是万众期待的iPhone 5。在自身出现问题，又遇到强大对手的情况下，HTC One X的惨败，似乎也在情理之中。　　　　一个人的成功除了自身的努力，也一定离不开历史的进程。也就是说，一样产品，如果设计理念过于超前，基本是很难成功的。Windows 8的扁平化风格可能就是很好的代表。幸运的是，Windows等到了扁平化普及的时代，因此得以正名。　　站在今天的角度来看，HTC One才是真正的里程碑式作品。全金属机身，双扬声器，UltraPixel相机，这些都是领先时代的。看看同期的竞品，还真是没有一个能打的。要知道，在HTC One之前，由于HTC One X的失利，HTC已经持续六个季度利润下滑。按照HTC官方的想法，这样一款功能超前而又极具美感的产品，一定能带领HTC重回巅峰的。　　可谁能想到，HTC One会因为主打的UltraPixel镜头，而被同期的三星Galaxy S4一脚踢翻。　　在当时，无论HTC将UltraPixel解读的多么厉害，都无法让用户信服400万像素能够拍出优秀的照片。当时用户并不懂，他们更愿意相信1300万像素一定比400万像素来的更让人放心。　　在2013年余下的时间里，HTC推出了HTC One mini与HTC One Max。值得一提的是，HTC One Max是第一款配备了指纹识别的安卓手机。其优秀的背部设计，被无数国产品牌所借鉴，沿用至今。　　HTC One不仅只有相机问题。当时由于市面上只有HTC一家在做全金属外壳手机，工艺难度较高。不仅费时费力，还受代工厂水平的制约。事实上，HTC One的工艺质量根本得不到保证，就算已经毁掉了大部分残次品，市场中依旧存在大量机身缝隙处能够塞进纸片的产品。　　一时之间，这部本应当拯救HTC的产品，却将HTC又一次拉进了深渊。　　　　2014年，HTC推出了One系列的新作，HTC One M8。M8是M7的完美升级版，机身设计更加圆润。尽管依旧坚持着全金属机身设计，但由于同年iPhone 6的发布，使得产业链整体迎来了升级。相较于前代M7，M8在品控上有了不小的进步。　　前代饱受诟病的400万像素摄像头，也被HTC升级为双摄像头。其中400万的主摄像头没有变化，加入了一颗全新的200万像素景深摄像头，使得M8在营销中占了上风。同时，老对手三星由于Galaxy S5遭遇滑铁卢，HTC终于迎来了久违的盈利。　　当然，HTC的盈利自然有M8的功劳，却也离不开Desire 816的助攻。这样一款性价比机器，帮助HTC成功杀入中端市场。　　但在2015年，市场又一次风云突变。　　M8的成功一度让HTC看到了希望，因此他们更进一步，推出了One系列的第三款产品，HTC One M9。M9在设计上更加注重细节，撞色工艺的运用使得整机更显华丽。400万像素的UltraPixel镜头也被移到手机正面，背部用上了常规的2000万像素摄像头。可以说，HTC One M9，是One系列完美的收官之作。　　但消费者显然不这么想。　　HTC One M9，并没有解决One系列饱受诟病的四下巴问题。一层叠一层的底栏，在观感上实在是让人厌恶。在三星Galaxy S6 edge的衬托下，HTC One M9的边框显得如此突出。设计与功能均不如竞品的情况下，HTC One M9的失败是注定的。　　但HTC的2015年并没有想象中那么失意，得益于一款像极了iPhone的产品，HTC A9的大卖，HTC勉强保住了颜面。　　尽管HTC A9的大卖让HTC迎来了短暂的春天，但很明显，HTC的市场份额依旧在不断的萎缩。高端市场打不过苹果三星，中低端市场又竞争不过小米，OPPO等厂商。HTC的未来，绝对算不上乐观。　　在这样的背景下，HTC 10登场了。　　HTC 10是一款听从消费者意见而推出的产品。它依旧采用了全金属机身。正面取消了HTC Logo与屏幕内导航键，十分简洁。无论是5.2英寸的2K分辨率屏幕，还是后置的1200万像素UltraPixel镜头，均显示出了HTC的诚意。　　　　2017年，HTC经历了人员变动，M系列成为了历史，U系列产品登上了舞台。　　但是，年初的HTC U Ultra，并没有带来HTC期望中的成功。相反，不久前刚刚发布的HTC U11却受到了一众媒体的好评。U11是一款配置强劲，各方面均衡产品。其拍照能力成功在DxOMark登顶。合理的价格，也让HTC U11成为了一款真正值得消费者考虑的产品。　　但HTC U11究竟能否让HTC重新起航，那就是只能交给时间去验证了。　　在手机业务萎靡的当下，HTC在VR领域取得了极大的成功。未来智能手机与VR之间能够产生哪些联动，或许才是能够帮助HTC手机业务复兴的关键。当然，这也可能是HTC能否存活的关键。　　　　其实HTC并没有做错什么，它每年都会推出一款不错的产品。但问题在于，市场已经变了，玩法已经变了。当苹果拿出App Store整合了软硬件资源，当小米开始思考如何布局智能家居，当三星整合了硬件供应链的时候，HTC这个一步一个脚印的老实人，注定是要掉队的。　　HTC被这个时代淘汰了。但仔细想想，诺基亚，摩托罗拉和黑莓，又有哪个不是呢？
