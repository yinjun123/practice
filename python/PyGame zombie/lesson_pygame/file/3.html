虾咩是Pygame？

大战前的宁静，是用来做好准备工作的！

游戏策划（GD、Game Designer）是游戏开发公司中的一种职称，是电子游戏开发团队中负责设计策划的人员，是游戏开发的核心。主要工作是编写游戏背景故事，制定游戏规则，设计游戏交互环节，计算游戏公式，以及整个游戏世界的一切细节等。

在本章里，主要涉及到本游戏的数值策划、资源文件，以及游戏逻辑策划。
