虾咩是Pygame？

一个简单的Pygame程序

我们先从一个简单的程序着手，这样能很快掌握一个游戏的基本元素：

  • 创建Surface对象
  • 窗口的大小和标题
  • 图片的加载和转换
  • 事件的获取和触发

import pygame, sys
#导入pygame库
from pygame.locals import *
#导入一些pygame常用的函数和常量

# 设置窗口大小
WINDOWWIDTH = 1024
WINDOWHEIGHT = 600

#初始化pygame,为使用硬件做准备
pygame.init()

# 创建了一个Surface窗口对象，大小为设置的宽度和高度
# 这个Surface对象以后会经常用到，我们可以在上面进行绘制其他东西
windowSurface = pygame.display.set_mode((WINDOWWIDTH, WINDOWHEIGHT))#, pygame.FULLSCREEN)

# 设置窗口标题
pygame.display.set_caption('豌豆射手大战妖艳贱货')

# 加载背景图片，获取图片对象
backgroundImage = pygame.image.load('resources\\background.png')

# 加载豌豆射手图片 并获取图片对象
playerImage = pygame.image.load('resources\\SnowPea.png')

# 将背景图片对象绘制到窗口上，并全窗口覆盖
windowSurface.blit(backgroundImage, (0, 0))

# 将豌豆射手绘制到窗口上，位置为： 窗口中间 和 下方 -70像素
windowSurface.blit(playerImage, (WINDOWWIDTH / 2, WINDOWHEIGHT - 70))

#游戏主循环
while True:
    #接收到退出事件后退出程序
    for event in pygame.event.get():
        if event.type == QUIT:
            sys.exit()

    #刷新屏幕，重新绘制画面
    pygame.display.update()

一个简单的Pygame程序就诞生了，后面我们会增加一些新的有趣的功能
