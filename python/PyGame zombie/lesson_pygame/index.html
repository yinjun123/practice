虾咩是Pygame？

Pygame的历史

  • Pygame是一个利用SDL库的写就的游戏库（游戏引擎），SDL呢，全名Simple DirectMedia Layer，是一位叫做Sam Lantinga的大牛写的，据说他为了让Loki（致力于向Linux上移植Windows的游戏的一家大好人公司，可惜已经倒闭，唉好人不长命啊……）更有效的工作，创造了这个东东。

  • SDL是用C写的，它也可以使用C++进行开发，当然还有很多其它的语言，比如Pygame就是Python中使用它的一个库。Pygame已经存在很多时间了，许多优秀的程序员加入其中，把Pygame做得越来越好。

安装Pygame

  • 你可以从www.pygame.org下载Pygame，选择合适你的操作系统和合适的版本，然后安装就可以了。你也可以通过Python的pip工具来安装(本文默认使用Python 3.5版本)：
pip3 install pygame
  • 一旦你安装好，你可以用下面的方法确认下有没有安装成功：
>>> import pygame
>>> print(pygame.ver)
1.9.2b6

  • 你的版本可能和我不同，这没关系。真正的知识是不会过时的，这里主要是依靠Pygame来介绍的游戏开发的方方面面，并不是说咱就可以靠这个做出什么伟大的游戏了（当然也不是说不可以）！

  • 就引擎本身而言，Pygame更致力于2D游戏的开发，也就是说，你可以用Pygame写一个植物大战僵尸，飞机大战等等，但是写一个魔兽世界则相当困难……

  • 但是请不要做出鄙夷的目光，底层的东西永远是相通的，而且对于新手而言，从简单的2D入手才是正途。

  • 并且就Python语言本身特性来说，Pygame还是非常易学易用的！

那我们就准备进入Pygame的世界吧！
